## 深浅拷贝的区别

### 我的回答
- 1. 深浅拷贝是针对于引用数据类型的，深拷贝像是克隆一样，将原有的数据进行克隆，克隆出来的数据改动不会影响原有的数据，浅拷贝是拷贝出来的数据改动会影响到原有的数据。
